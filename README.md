# BRN PR Calendar V2

เว็บแอปหน้าเปิดประจำวันของทีมประชาสัมพันธ์เทศบาลเมืองบางรักน้อย

แกนหลักของเว็บ:

- วันนี้ทีมควรสื่อสารอะไร
- เดือนนี้มีวันสำคัญและธีมคอนเทนต์อะไร
- ปฏิทินงานเทศบาลและวันสำคัญ
- บอร์ดสถานะงาน PR
- คัมภีร์ PR ตามวิสัยทัศน์ใหม่
- Prompt สำหรับนำไปใช้ต่อในทันใจ AI Studio
- ฉาก 3D ด้วย Three.js แบบ local ไม่พึ่ง CDN ตอนใช้งานจริง และเปิดจากไฟล์ `index.html` ได้
- โทนม่วงทองเทศบาล + ส้ม PR

## เปิดใช้งานในเครื่อง

```bash
node server.mjs
```

แล้วเปิด `http://localhost:4173`

## ใช้กับ GitHub Pages

โปรเจกต์นี้เป็นเว็บ static ใช้ไฟล์หลักคือ `index.html`, `styles.css`, `app.js`, `scene.js` และโฟลเดอร์ `assets/`

วิธีขึ้น GitHub Pages:

1. สร้าง repository ใหม่บน GitHub
2. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้
3. ไปที่ Settings > Pages
4. เลือก Deploy from a branch
5. เลือก branch `main` และ folder `/root`

## ข้อมูลที่เก็บในเครื่อง

งานที่เพิ่มในหน้าเว็บจะเก็บด้วย `localStorage` ของ browser เครื่องนั้นก่อน เหมาะสำหรับต้นแบบและทดสอบ flow ทีม
