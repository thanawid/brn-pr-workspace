# BRN PR Content Calendar

เว็บแอปสำหรับทีมประชาสัมพันธ์เทศบาลเมืองบางรักน้อย เปิดดูทุกวันเพื่อวางแผนว่า “วันนี้ต้องเล่าอะไร” และ “เดือนนี้ควรทำคอนเทนต์ช่วงไหน”

## สิ่งที่มี

- หน้าแรกใช้ปฏิทินคอนเทนต์เป็นพระเอกเต็มพื้นที่
- โทนสีม่วง ส้ม เทา แบบเครื่องมือทีม PR ไม่ฟุ้งเกินไป
- แถบนำทางแนวนอน และการ์ดปฏิทินสะอาดขึ้น
- มาสคอต “น้องคอนเทนต์” อยู่ในส่วนผู้ช่วย ไม่แย่งพื้นที่ปฏิทิน
- PR Assistant สำหรับตอนทีมคิดคอนเทนต์ไม่ออก
- การ์ดไอเดียคอนเทนต์ที่บอกมุมเล่าเรื่อง ภาพนิ่ง วิดีโอ และ workflow
- กดไอเดียเพื่อเพิ่มลงปฏิทิน หรือส่งไปสร้าง Prompt ได้ทันที
- AI Radar แนะนำว่าวันนี้ควรทำคอนเทนต์อะไร ควรถ่ายอะไร และควรเดินงานแบบไหน
- ฉาก 3D เป็นบรรยากาศรอง ส่วนปฏิทินยังเป็นพระเอก
- ปฏิทินปี 2569 พร้อมวันสำคัญและวันพระไทย
- กดวันที่ในปฏิทินเพื่อเปิด Day Board รายวัน
- เพิ่มงานเทศบาล วัน/กิจกรรม โน้ตทีม และคอนเทนต์ไอเดียได้เอง
- บอร์ดสถานะงาน: ไอเดีย, รอถ่าย, ตัดต่อ, รออนุมัติ, เผยแพร่แล้ว
- ตัวช่วยคิดมุมคอนเทนต์รายเดือนและรายวัน
- Prompt สำหรับนำไปใช้ต่อใน ทันใจ AI Studio
- ไม่ใช้โลโก้เดิมในหน้าเว็บ มีไอคอนชั่วคราวของระบบแทน

## วิธีเปิดใช้งาน

เปิด `index.html` ได้โดยตรง หรือรันเซิร์ฟเวอร์ในเครื่อง:

```bash
node server.mjs
```

แล้วเปิด `http://localhost:4173`

## ข้อมูลที่เพิ่มเอง

งานที่ทีมเพิ่มในหน้าเว็บจะเก็บไว้ใน `localStorage` ของ browser เครื่องนั้น เหมาะสำหรับต้นแบบและทดสอบ flow ทีม ถ้าจะให้หลายคนใช้ข้อมูลชุดเดียวกันจริง ค่อยต่อระบบฐานข้อมูลภายหลังได้

## ที่มาวันพระ

รายการวันพระ พ.ศ. 2569 อ้างอิงจาก MyHora:  
https://www.myhora.com/calendar/buddhist-2569.aspx

ควรตรวจสอบอีกครั้งกับประกาศหรือปฏิทินราชการที่หน่วยงานใช้ ก่อนนำไปใช้กับงานพิธีการที่ต้องการความถูกต้องสูง

## ใช้กับ GitHub Pages

โปรเจกต์นี้เป็นเว็บ static ใช้ไฟล์หลักคือ `index.html`, `styles.css`, `app.js`, `scene.js` และโฟลเดอร์ `assets/`

วิธีขึ้น GitHub Pages:

1. อัปโหลดไฟล์ทั้งหมดเข้า repository
2. ไปที่ Settings > Pages
3. เลือก Deploy from a branch
4. เลือก branch `main` และ folder `/root`
