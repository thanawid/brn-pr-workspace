# BRN PR Board V1.0 — Audit จากฐาน V1.6

## สิ่งที่มีอยู่แล้วและเก็บไว้
- ปฏิทินงานภาษาไทย
- วันสำคัญและวันพระ
- ฟอร์มเพิ่ม/แก้ไขงานเดิมทั้งหมด
- เวลาเริ่ม เวลาสิ้นสุด งานตลอดวัน สถานที่ เจ้าของงาน และประเภทงาน
- ค้นหาและกรองงาน
- แจ้งกลุ่ม LINE เมื่อบันทึก
- Firebase Authentication และ Firestore
- การเรียงงานรายวัน
- คำแนะนำภาพ วิดีโอ การเตรียมงาน และคอนเทนต์

## สิ่งที่ขาดและเพิ่มใน V1.0
- หน้า AI ผู้ช่วยงาน PR ที่อ่านข้อมูลจากวันที่/งานที่เลือกจริง
- สรุปบริบทของงาน เจ้าของงาน และสถานที่ก่อนแสดงคำแนะนำ
- คำแนะนำ 5 ส่วนตามแนวคิดตั้งต้น:
  1. วันนี้ต้องเตรียมอะไร
  2. ภาพควรเป็นแบบไหน
  3. ควรเก็บคลิปอะไร
  4. คิดคอนเทนต์อะไรได้บ้าง
  5. เดือนนี้ทำอะไรดี
- แผนประจำเดือนรวมวันสำคัญและงานจริงในปฏิทิน
- Avatar สำรองสำหรับบัญชี PR01–PR05 เมื่อไม่มีรูปจริง

## สิ่งที่ปรับออก
- หน้าแนวทางแบบคู่มือยาวคงที่ ถูกเปลี่ยนเป็นหน้าแนะนำตามงานจริง

## สิ่งที่ไม่แตะ
- Firebase project `brn-pr-board`
- Authentication contract
- Firestore collection `prEvents`
- LINE queue `lineOutbox`
- document `system/lineGroup`
- Cloud Functions `lineWebhook`, `sendLineOutbox`
- Eventarc และระบบหลังบ้านทั้งหมด

## หลักถาวร
งานในปฏิทินเป็นศูนย์กลาง และ AI ทำหน้าที่อ่านข้อมูลงานเพื่อช่วยเตรียมภาพ คลิป คอนเทนต์ และแผนงานประชาสัมพันธ์
