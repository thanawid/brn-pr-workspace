# BRN PR Board Project Lock

เอกสารนี้ล็อกทิศทางของระบบ เพื่อไม่ให้หลุดจากเป้าหมายหลักของทีมประชาสัมพันธ์เทศบาลเมืองบางรักน้อย

## หลักที่ต้องรักษาไว้
- ปี พ.ศ.
- ชื่อเดือนและวันภาษาไทย
- วันพระในช่องปฏิทิน
- วันหยุดและวันสำคัญ
- สีแยกประเภทระหว่างวันสำคัญ วันพระ และงานทีม
- การเลือกวันแล้วลงงาน
- ข้อมูลวันต่าง ๆ สำหรับวางแผนงานล่วงหน้า

## สิ่งที่หน้าเว็บต้องเป็น
- ปฏิทินใหญ่เต็มพื้นที่ เป็นกระดานงานหลัก
- ไม่มีแถบรายละเอียดด้านขวาค้างไว้
- คลิกวันที่แล้วเปิดหน้าต่างเพิ่มงาน
- ฟอร์มงานมีชื่องาน รายละเอียด วัน เวลา สถานที่ และเจ้าของงาน
- ไม่ต้องมีผู้รับผิดชอบรายคน เพราะเป็นทีม PR เดียวกัน
- งานที่บันทึกต้องแสดงในปฏิทินและใช้ร่วมกันทุกอุปกรณ์
- ใต้ปฏิทินมีการ์ดแนวทางถ่ายภาพ ถ่ายคลิป สิ่งที่ควรเตรียม และคอนเทนต์แนะนำ
- มีคำแนะนำประจำเดือนและวันสำคัญ เผื่อทีมคิดงานไม่ออก
- ตัวช่วยคิดคอนเทนต์อยู่เบื้องหลัง ไม่ต้องเป็นตัวนำของหน้าเว็บ
- มี animation ได้ แต่ต้องไม่รบกวนการใช้งาน
- ไม่มีมาสคอต
- ไม่มีเลขเวอร์ชันโชว์หน้าเว็บ
- รองรับมือถือ แท็บเล็ต โน้ตบุ๊ก และจอใหญ่

## หลังบ้านที่ห้ามเปลี่ยน
- Firebase project `brn-pr-board`
- Authentication
- Firestore
- Cloud Functions
- `lineWebhook`
- `sendLineOutbox`
- `system/lineGroup`
- `lineOutbox`
- การแจ้งเตือนเข้ากลุ่ม LINE ที่ทดสอบผ่านแล้ว

ทุกการแก้ไขหลังจากนี้ต้องยึด “งานนำ ปฏิทินนำ ตัวช่วยอยู่หลังฉาก” เป็นแกนหลัก
