# Implementation Notes — V1.2.7

- ใช้ V1.2.2 เป็นฐานและคง Firebase, Firestore, LINE, ปฏิทิน, ใบงาน และการพิมพ์เดิม
- เรียงเมนูเป็น ปฏิทิน → ภาพรวม → งานของทีม และให้ปฏิทินเป็นจุดหลักของระบบ
- กดโลโก้ BRN PR Board แล้วกลับไปยังปฏิทิน
- เปลี่ยนโลโก้และ favicon เป็นไอคอนกล้อง–ปฏิทินสีม่วง–ทอง
- ปรับกล่องงานถัดไปใน Hero เป็นกระจกฝ้าโปร่งแบบเดิม
- งานใกล้ถึงที่ควรเตรียมแสดง 3 งาน พร้อมปุ่มดูงานทั้งหมด
- เปลี่ยนการ์ดแผนเตรียมงานเป็น “เดือนนี้สื่อสารอะไรดี?” พร้อมไอเดีย 3 รายการ
- ไอเดียประจำเดือนใช้ข้อมูลงานในปฏิทิน วันสำคัญ และแนวคิดทั่วไป โดยไม่สร้างข้อเท็จจริงราชการเอง
- เพิ่มปุ่มดูไอเดียทั้งหมดและคิดไอเดียใหม่
- เปลี่ยนคำในใบงานจาก “ระบบสนับสนุน PR” เป็น “แนวทางงาน PR”
- อัปเดต cache bust, Service Worker และ PWA icons เป็น 1.2.3

- เพิ่มกลิ่นอายปฏิทินไทยแบบ “ไทยกำลังดี”: วันพระมีเงาพระจาง ๆ พร้อมข้างขึ้น–ข้างแรม และวันสำคัญมีไอคอนมุมช่องแบบสุภาพ

- ปรับสัญลักษณ์ปฏิทินเป็นไอคอนลายเส้นลงสีอ่อนแบบชอล์ก/พาสเทล โดยคงความอ่านง่ายของงานในแต่ละช่อง

- เปลี่ยนสัญลักษณ์ปฏิทินจากไอคอนวาดในโค้ดเป็นชุดภาพประกอบเล็กแบบไทยร่วมสมัย และลดความเด่นของวันที่อยู่นอกเดือนปัจจุบัน

- อัปเกรดชุดภาพประกอบวันสำคัญและวันพระให้เป็นภาพประกอบจริงมากขึ้น (ไม่ใช่เส้นไอคอนล้วน) พร้อมลดความเป็นป้ายกรอบของภาพมุมปฏิทิน
