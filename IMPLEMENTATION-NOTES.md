# Implementation Notes — V1.2.1

ฐาน: V1.2.0

## การเปลี่ยนแปลง
- แยกหน้า “ภาพรวม” และ “ปฏิทิน” ด้วย main view state โดยไม่รื้อ renderer ปฏิทิน
- ปฏิทินถูกซ่อนในหน้าเริ่มต้นและแสดงเมื่อเลือกเมนูปฏิทิน
- เพิ่มภาพประกอบใน Hero และส่วนวันสำคัญจาก assets เดิม
- เพิ่ม motion แบบเบาและ fallback สำหรับผู้ใช้ที่ลดการเคลื่อนไหว
- อัปเดต cache bust และ service worker เป็น 1.2.1

## สิ่งที่คงเดิม
