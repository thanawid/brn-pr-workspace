# Implementation Notes — V1.2.0

## แนวคิดการออกแบบ

ระบบยังยึด “งานนำ” เป็นแกนหลัก แต่เพิ่มความน่าใช้ด้วยฟอนต์ไทยที่อ่านง่าย ภาพประกอบแบบเบา สีสถานะ ระยะห่าง และ responsive โดยไม่เปลี่ยน workflow เดิม

## สิ่งที่ทำในรอบนี้

1. เพิ่มแบบแจ้งงานประชาสัมพันธ์ที่บันทึกลง `prEvents` โดยตรง
2. คงปุ่มเพิ่มงานเองของทีม PR ไว้เหมือนเดิม
3. เพิ่ม `requestSource: "request_form"` และ `prTasksOther` เป็นฟิลด์ optional
4. ปรับ Typography ด้วย Noto Sans Thai และ Anuphan พร้อม line-height และการตัดคำไทย
5. เพิ่มภาพประกอบใน Hero และแบบแจ้งงานจาก asset เดิม
6. ปรับ Toast, Footer, ลิขสิทธิ์ และสีสันเฉพาะจุด
7. ปรับใบสรุป A4 ให้เว้นชื่อผู้บันทึกเมื่อบัญชีเป็นชื่อทั่วไป เช่น “เจ้าหน้าที่ PR 01”

## แนวทางรอบถัดไป

- ทดสอบกับทีมจริงและเก็บบั๊กจากมือถือ/เครื่องพิมพ์
- เพิ่ม cover image/thumbnail เฉพาะงานที่มีสื่อแล้ว เมื่อ workflow การแนบไฟล์ชัดเจน
- เชื่อมปุ่มส่งบรีฟไป Tanjai AI Studio เมื่อกำหนดสัญญาข้อมูลระหว่างสองระบบแล้ว

ไม่ควรเพิ่มกราฟหนัก Sidebar ซับซ้อน หรือ AI Chat เต็มหน้า เพราะจะทำให้แกนงานอ่อนลง
